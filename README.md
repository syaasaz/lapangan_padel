## 📖 Tema Kasus
Aplikasi ini menggunakan tema **Sistem Informasi Reservasi Lapangan** berbasis web menggunakan framework Laravel.

Sistem ini dibuat untuk mempermudah proses pemesanan lapangan serta membantu pengelola dalam mengelola data reservasi secara lebih efektif.

---

## 🟨 Latar Belakang
Proses reservasi lapangan olahraga di beberapa tempat masih dilakukan secara manual, seperti melalui pencatatan buku atau pesan singkat, sehingga sering terjadi kesalahan pencatatan dan bentrok jadwal. Selain itu, pengguna juga kesulitan mengetahui ketersediaan lapangan secara cepat dan akurat. Oleh karena itu, diperlukan sistem informasi reservasi lapangan berbasis web yang dapat mempermudah proses pemesanan, menyediakan informasi jadwal secara real-time, serta membantu pengelola dalam mengelola data secara lebih efektif dan terstruktur.

---

## 🎯 Tujuan
Tujuan dari sistem ini adalah untuk mempermudah proses reservasi lapangan secara online, menyediakan informasi jadwal secara real-time, serta membantu pengelola dalam mengelola data pemesanan secara efektif dan terstruktur.
