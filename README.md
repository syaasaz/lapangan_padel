## 📖 Tema Kasus
Aplikasi ini mengangkat tema **Sistem Informasi Reservasi Lapangan** berbasis web menggunakan framework Laravel.

Sistem ini dibuat untuk mempermudah proses pemesanan lapangan serta membantu pengelola dalam mengelola data reservasi secara lebih efektif.

---

## 🟨 Latar Belakang
Proses reservasi lapangan masih dilakukan secara manual sehingga sering terjadi kesalahan pencatatan dan bentrok jadwal, sehingga diperlukan sistem berbasis web untuk mempermudah pengelolaan dan pemesanan.

---

## 🎯 Tujuan
Tujuan dari sistem ini adalah untuk mempermudah proses reservasi lapangan secara online, menyediakan informasi jadwal secara real-time, serta membantu pengelola dalam mengelola data secara efektif.
